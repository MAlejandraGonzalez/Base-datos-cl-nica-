# Base de Datos - Historias Clínicas

Este proyecto contiene el script SQL para la creación de una base de datos de una clínica.

## Incluye:
- Tablas: Pacientes, Médicos, Turnos, Historias Clínicas
- Relaciones con claves foráneas
- Datos de prueba

## Cómo usar:
Ejecutar el archivo .sql en SQL Server Management Studio.
